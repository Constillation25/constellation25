# Constellation25 — Polyglot Multi-LLM Agent Framework

## What is Constellation25?
A planetary agent orchestration system where named celestial body agents
are assigned to specific LLMs based on trained capabilities.

## Core Concept
- Each agent = a planet or celestial body
- Each LLM = a specialized capability
- Agents route tasks to the right LLM automatically

## LLM Role Assignment
Claude    → architecture, reasoning, code review
ChatGPT   → generation, iteration  
Grok      → real-time context, X/Twitter
Qwen      → multilingual, alternative perspectives
Ollama    → free local inference
